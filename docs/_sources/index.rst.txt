hicsv-python documentation
==========================

Python implementation of the I/O interface 
for the Header-Included Comma-Separated Values 
(`hicsv`) file format. 

Installation
------------

conda

.. code-block:: console

   conda install -c conda-forge hicsv-python

pip

.. code-block:: console

   pip install hicsv-python
